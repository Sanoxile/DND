The [[Fighter]] class purpose is kind of strange. It blends almost everything someone could want to do. Bows, swords and sometimes spells. A blend of them all. This class offers a lot of variety. Currently it is strength and constitution focused. The plan is to let the user chose to be proficient in dex or strength based on the subclass. This class will be built around this new idea. Also the fighter is strong to start in DND but weak later. This rework will aim to fix all that.
